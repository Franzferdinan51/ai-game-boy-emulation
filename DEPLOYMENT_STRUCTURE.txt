## EXCLUDED FILES (DO NOT INCLUDE IN DEPLOYMENT)
- ❌ All *.log files (development logs)
- ❌ All __pycache__ directories
- ❌ All node_modules directories
- ❌ All .venv directories
- ❌ All .crush directories
- ❌ All temporary files (*.tmp, *.temp)
- ❌ Development test files (test_*.py, *_test.py)
- ❌ Multiple similar startup scripts (keep only unified_startup.*)
- ❌ Documentation files not essential for deployment
- ❌ Development configuration files
- ❌ Build artifacts and cache files
- ❌ User-specific configuration files
- ❌ Archive files (*.zip, *.tar, *.gz)

## DEPLOYMENT PACKAGE SIZES (ESTIMATED)
- Core System (Backend + Frontend): ~15-20 MB
- With PyBoy Emulator: ~25-30 MB
- With PyGBA Emulator: ~30-35 MB
- Complete Package: ~35-40 MB

## MINIMUM REQUIREMENTS FOR FUNCTIONAL SYSTEM
- Essential: ai-game-server/ + ai-game-assistant/ + unified_startup.bat
- Recommended: Essential + PyBoy/ + install_dependencies.bat
- Complete: Recommended + PyGBA + all documentation

## DEPLOYMENT INSTRUCTIONS
1. Copy all listed files to the target system
2. Run install_dependencies.bat to install required packages
3. Run unified_startup.bat to start the system
4. Access the web interface at http://localhost:5173

## PORTABLE DEPLOYMENT OPTION
For a truly portable deployment that doesn't require internet:
- Include node_modules/ directory (adds ~100MB)
- Include Python virtual environment (adds ~200MB)
- Total portable size: ~335-340 MB
